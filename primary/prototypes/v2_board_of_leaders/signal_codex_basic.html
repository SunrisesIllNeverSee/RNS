Signal Codex

Big 3 Metrics

  • Transmitter Composition (T): Weighted clarity across 5 metrics
    Formula: 0.30×CR + 0.25×PC + 0.20×CT + 0.10×SD + 0.10×TT + 0.05×MV
  • SDRM: Signal integrity over time
    Formula: CR × (SD + PC) × Thread Rate
  • Signal Force (SF): Sustained throughput and resilience
    Formula: (Total Msgs × Avg Session) ÷ Account Age

Core Metrics

  • CR: Compression Ratio
  • PC: Prompt Complexity
  • CT: Cross-Thread Referencing
  • SD: Session Depth
  • TT: Token Throughput
  • MV: Message Volume
