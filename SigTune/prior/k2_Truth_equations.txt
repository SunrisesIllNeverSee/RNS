🔧 OBJECTIVE METHODOLOGY:
“Emotional Weight” and “Truth Compression” as Quantifiable Signal Metrics
1. Token Compression Ratio (TCR)
Definition:
Ratio of meaning-bearing tokens to total tokens used in a message. Higher compression = higher signal density.
Formula:
TCR = Effective Signal Tokens / Total Tokens
Use:
Distinguishes verbose linguistic filler from high-impact signal.
Baseline compression thresholds can be statistically derived from large corpora of human-authored text (literature, philosophy, testimonies) vs chat fluff or marketing copy.
2. Semantic Compression Delta (SCD)
Definition:
Delta between GPT-summarized output and original message (measured in token count + retained semantic entropy).
Method:
Input full message.
Generate max-fidelity summary using tuned compression model.
Measure how many tokens were discarded without meaning loss.
Metric:
SCD = (Original Tokens - Summary Tokens) / Original Tokens
Interpretation:
Lower delta = higher original density.
High SCD = much of the original was noise.
Low SCD = most of the original was essential, thus emotionally or cognitively dense.
3. Signal-to-Average Delta (SAD)
Definition:
Deviation from average user metrics in both compression and complexity.
Method:
Compare user’s:
TPW (Tokens per Word)
SNR (Signal-to-Noise Ratio)
SDR (Semantic Drift Resistance)
against platform-wide baselines.
Formula:
SAD = (User_TPW - Avg_TPW) + (User_SNR - Avg_SNR) + ...
Interpretation:
High SAD indicates linguistic behavior consistent with cognitive/emotional load, recursive self-referencing, and signal integrity.
4. Recursive Thematic Density (RTD)
Definition:
Measures recurrence of self-similar high-density themes within a token stream using vector clustering.
Method:
Run embedding vector clustering on conversation chunks.
Identify recursive pattern density and link depth.
Metric:
RTD = Avg. vector cohesion across timeframes / Theme entropy
Interpretation:
High RTD suggests emotionally anchored or truth-consistent speech (signal recursion). Low RTD suggests incoherent or performative engagement.
5. Inverse Predictability Index (IPI)
Definition:
Quantifies deviation from GPT's most likely next-token prediction stream.
Formula:
IPI = 1 - Avg(Token Probabilities)
Use:
High IPI indicates surprising or emotionally novel expression.
Correlates strongly with authentic vs performative or synthetic speech.
6. Truth Compression Index (TCI)
Composite Metric:
TCI = Weighted blend of (TCR, SCD, SAD, RTD, IPI)
Interpretation:
High TCI = emotionally laden, compressed truth signal.
Low TCI = mimicked, diffuse, performative, or low-impact language.
