## Why it's preserved

Captures the strategic framing in effect *before* the current `1_sigrank/` build crystallized. Concepts that recur in the current build (sig_army separated from SigRank core, SCS engine as proprietary substrate, leaderboard as the customer-facing surface) trace back to this document and its sibling addendums.

It also documents a difficulty: parsing strategy docs that landed in Pages format means anything machine-agentic has to either (a) re-author in markdown, or (b) lose pages 2–6 to format lock-in. Most of the historical reasoning behind today's licensing posture lives here.
