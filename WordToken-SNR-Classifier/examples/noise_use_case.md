# Example: Noise Use

Sentence: "Basically, like, you know, it's just kind of... the thing we were doing."

In this case:
- 'basically', 'like', 'you know', 'just', 'kind of' → noise (low semantic contribution)
- 'thing', 'doing' → ambiguous; depends on context

Same token 'just' is noise here, but could be signal in emotional emphasis elsewhere.
